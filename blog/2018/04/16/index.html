15 April 2018, Neil Parkhi

Introduction

The Lexus LC is a new type of vehicle Lexus is offering. It is a very unique coupe in its very own way which is based off the Lexus LS sedan. It looks nice, feels nice, and has some of the greatest in terms of modern technology inside the vehicle.

Specifications

Steering and Transmission
Steering Type Electric Assist, Rack and Pinion Power Steering
Transmission Type Continuous Variable Automatic
Suspension
Front Suspension Type Double Joint Multi-Link
Rear Suspension Type Double Joint Multi-Link
Suspension Features Stabilizer Bars, Coil Springs, Linear-Solenoid Actuated Shock Absorbers
Dimensions
Length (in, mm) 187.4in, 4760mm
Width (in, mm) 75.6in, 1920mm
Height (in, mm) 53.0in, 1346mm
Wheelbase (in, mm) 113.0in, 2870mm
Weight (lb, kg) 4280lb, 1941kg
Engine Specifications
Engine Type 5.0 litre 32 valve double overhead cam V8
Power (bhp, PS, kW) 471bhp, 478PS, 351kW @ 7100RPM
Torque (lb-ft, Nm) 398lb-ft, 540Nm @ 4800RPM
0-60 mph (0-97 kmph) 4.4 seconds
Top Speed (mph, kmph) 168 mph, 270 kmph
Drivetrain Rear Wheel Drive
Fuel Tank Size (gal, L) 21.7 gallons, 82L
Fuel Type (AKI, RON) 91 Octane/AKI Petrol (95 RON)
City Fuel Economy (mpg, kmpL) 16mpg, 7kmpL
City Fuel Economy (mpg, kmpL) 26mpg, 11kmpL
Cargo Specifications
Cargo Space (ft3, L) 5.4ft3, 153L
